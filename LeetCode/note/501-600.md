# LeetCode 501-600



## 509.斐波那契数(简单)

https://leetcode.cn/problems/fibonacci-number/description/

```c
int fib_arr[31] = {0, 1};

int fib(int n){
    int fact1, fact2;

    if(n < 2) {
        return n;
    }
    else {
        fact1 = fib_arr[n-1] > 0 ? fib_arr[n-1] : (fib_arr[n-1] = fib(n-1));
        fact2 = fib_arr[n-2] > 0 ? fib_arr[n-2] : (fib_arr[n-2] = fib(n-2));
    }

    return fact1 + fact2;
}
```
