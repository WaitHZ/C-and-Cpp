## 374.猜数字大小

https://leetcode.cn/problems/guess-number-higher-or-lower/description/

```c
int guessNumber(int n){
	int left, right, mid;

    left = 1;
    right = n;

    while(left <= right) {
        mid = (right - left) / 2 + left;

        if(guess(mid) == 0)
            break;
        else if(guess(mid) == 1)
            left = mid + 1;
        else
            right = mid - 1;
    }

    return mid;
}
```
