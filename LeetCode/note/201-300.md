# LeetCode 201-300



## 278.第一个错误的版本

https://leetcode.cn/problems/first-bad-version/description/

```c
int firstBadVersion(int n) {
    int left, right, mid, retVal;

    left = 1;
    right = n;

    if(isBadVersion(1))
        retVal = 1;
    else {
        while(left <= right) {
            mid = (right - left) / 2 + left;

            if(!isBadVersion(mid))
                left = mid + 1;
            else {
                if(!isBadVersion(mid-1)) {
                    retVal = mid;

                    break;
                }
                else
                    right = mid - 1;
            }
        }
    }

    return retVal;
}
```

需要注意，我们通过下方的代码来避免溢出：

```c
mid = (right - left) / 2 + left;
```

而不采用：

```c
mid 
```
